# Bowlr

A modern web application for creating and managing lawn bowls tournaments.

## Features

- **Tournament Creation**: Easily set up new lawn bowls tournaments with customizable formats
- **Player Management**: Register and organize participants
- **Scheduling**: Automated match scheduling and round management
- **Live Scoring**: Real-time score tracking and updates
- **Results Management**: Record match outcomes and generate final standings
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone <repository-url>
cd bowlr
npm install
```

### Running the Application

```bash
npm start
```

The application will be available at `http://localhost:3000`

## Usage

1. Create a new tournament
2. Add players to your tournament
3. Generate match schedule
4. Update scores during play
5. View final results and standings
